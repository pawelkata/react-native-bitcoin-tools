# react-bitcoin-tools

A set of helpers for interacting with Bitcoin and Blockchain.

## Installation

```sh
npm install react-bitcoin-tools
```

## Usage


```js
import { multiply } from 'react-bitcoin-tools';

// ...

const result = await multiply(3, 7);
```
